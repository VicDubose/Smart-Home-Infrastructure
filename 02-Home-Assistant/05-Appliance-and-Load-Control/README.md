# ⚡ Load Shaving & Appliance Control

## 🎯 Purpose

Load shaving ensures the home does not run multiple high-energy appliances simultaneously during expensive or constrained energy periods.

This system protects:
- Battery longevity  
- Cost efficiency  
- Grid independence  

---

## 🧠 Why Load Shaving Exists

The home operates as a battery-first energy system, not a grid-first system.

Unlike traditional homes:
- Power is finite (battery + solar)
- Timing matters more than total usage
- Multiple large loads can:
  - Drain the battery too quickly  
  - Force unnecessary grid usage  
  - Trigger higher utility costs  

---

## ⚡ Core Problem

Running multiple high-load devices at the same time causes:

- Rapid battery depletion  
- Loss of off-grid coverage  
- Increased dependency on grid power  
- Reduced system efficiency  

---

## 🔌 High-Impact Loads

This system actively manages:

- Washer  
- Dryer  
- Dishwasher  

These devices are considered:

> Flexible loads — meaning they can be delayed without impacting safety

---

## ⚙️ Load Shaving Strategy

### 1. Time-Based Enforcement (Alabama Power)

- CHEAP window → loads allowed  
- NORMAL / CAUTION → limited usage  
- STRICT (peak) → loads blocked  

---

### 2. Battery-Aware Behavior

Loads are restricted when:
- Battery must carry the home  
- Energy must be preserved for later windows  

---

### 3. Solar Utilization

When solar is available:
- Loads are encouraged to run  
- Energy is consumed locally instead of wasted  

---

### 4. No Load Stacking

System prevents:

- Washer + Dryer running together  
- Dishwasher running during peak  
- EV charging overlapping heavy loads  

---

## 🧠 Appliance Behavior

### Dishwasher

- Only allowed during CHEAP window  
- Immediately turned off if outside allowed time  

---

### Washer

- Allowed during savings window  
- If started outside window:
  - 15-second grace period  
  - Automatically paused  
  - User notified via Apple Home  

---

### Dryer (Planned / Partial)

- High consumption device  
- Restricted to CHEAP window only  

---

## 🔔 User Interaction Model

This system is designed to:

- Allow normal usage  
- Detect incorrect usage  
- Correct behavior automatically  
- Notify the user when intervention occurs  

---

## ⚡ Design Philosophy

- Prevent bad energy decisions automatically  
- Avoid requiring user awareness of time windows  
- Maintain comfort while optimizing cost  

---

## 🧠 System Impact

Load shaving directly supports:

- 15–17 hour off-grid operation  
- Reduced peak energy usage  
- Improved battery efficiency  
- Lower monthly utility costs  

---

## ⚡ Summary

Load shaving ensures energy is used:

- At the right time  
- By the right devices  
- Without overlapping demand  

This allows the home to operate as a coordinated energy system, not a collection of independent appliances.
